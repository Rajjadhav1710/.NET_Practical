
# .NET_Practical

This repo contains all the practicals done during the semester in which we have .NET as one of the elective. All practicals are implemented in c# language. practicals include concepts like basics of c#, variables, datatypes, branch and loop statements, string interpolation, various string operations, oops, functions, properties, indexers, generics, lambda expressions, delegates, collections and in last one asp.net core mvc project. All practicals are done under the guidance of our professor Meghna Desai mam.
